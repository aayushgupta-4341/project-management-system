## рҹ§  Project Overview

This project is a **full stack Project Management System** designed to help teams manage projects and tasks efficiently.

The application provides a structured way to:

* Create and manage projects
* Add and track tasks
* Monitor progress through a dashboard
* Authenticate users securely

It solves the problem of **unorganized task handling and lack of visibility** in small teams or startups.
